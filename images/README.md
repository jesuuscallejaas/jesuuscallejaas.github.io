# Carpeta de Imágenes

Esta carpeta contiene las imágenes utilizadas en el portfolio.

## Archivos recomendados:

- `profile.jpg` o `profile.png` - Tu foto de perfil (recomendado: 500x500px)
- `hero-bg.jpg` - Imagen de fondo opcional para la sección hero

## Formatos recomendados:
- JPG para fotografías
- PNG para logos o imágenes con transparencia
- WebP para mejor optimización (navegadores modernos)

## Tamaño recomendado:
- Foto de perfil: 500x500px a 800x800px
- Mantén las imágenes optimizadas (< 500KB cada una)
